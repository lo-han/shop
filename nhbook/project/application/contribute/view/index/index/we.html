关于我们

内涵书城，是衡阳热血文化传媒有限公司旗下为作家提供作品展示的专业平台。

公司有着强大的渠道资源，在版权代理方面，与国内多家大型文学网站及新媒体渠道，有着深厚的合作，能为签约作家展开电子版权、新媒体渠道、漫画、出版、影视作品等全方位运营。

目前公司和网站已经运营出多部全渠道畅销作品。
